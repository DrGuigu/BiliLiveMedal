# BiliLiveMedal - Bilibili直播 粉丝勋章工具
- 查询别人的粉丝勋章
- 直播间排行榜及亲密度对比 
- 查别人的关注(B站官方限制100, 程序可以查到250个, 更多的由于API限制查不到)

想整活的自行编译, 用了Unirest和Guava, Flatlaf就是为了好看, 去掉也可以
